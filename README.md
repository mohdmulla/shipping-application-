A simple python flask application for courier service provider. 

<b> installation <b> :
  make sure to use virtual environment   :
  
  installing libraries : 
  #install from requirements
  pip install -r requirments.txt 
  
  To run the app 
  export FLASK_APP = app.py
  python -m flask run
